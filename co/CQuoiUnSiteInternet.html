Définir un site internet

Définition :

Un site web, ou simplement site, est un ensemble de pages web et de ressources liées et accessible par une adresse web. Un site est hébergé sur un serveur web accessible via le réseau mondial internet ou un intranet local. L'ensemble des sites web constituent le World Wide Web.

Afin que les internautes puissent consulter le site internet, il faut que l'ensemble de votre site (page html, texte, images...) soit déposé sur un serveur (hébergeur définition

L'une des façons les plus rapides de trouver un site internet pour l'utilisateur est de connaître son nom de domaine (ex : www.monsite.fr) ou par la saisie de mot-clefs pertinents dans un moteur de recherche (référencement).

Aujourd'hui des outils permettent de créer un site internet sans forcément savoir coder, on appelle ces outils des CMS (Content Management System ou Gestionnaire de contenu), quelques exemples connus : WordPress, Joomla, Wix,Jimdo.
